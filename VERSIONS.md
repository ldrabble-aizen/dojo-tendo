# Versiones publicas

## Mini Kombat 1

https://ldrabble-aizen.github.io/dojo-tendo/

Este link corresponde a la raiz del sitio de GitHub Pages. Para mantenerlo como Mini Kombat 1, no hay que publicar nuevas versiones sobre `index.html`, `game.js`, `styles.css` y `assets/` de la raiz.

## Mini Kombat 2

Link previsto:

https://ldrabble-aizen.github.io/dojo-tendo/mini-kombat-2/

La version nueva vive en `mini-kombat-2/`. Cuando este terminada, se puede publicar sin romper el link de Mini Kombat 1.

## Regla para futuras secuelas

Cada nuevo juego debe ir en una carpeta nueva, por ejemplo `mini-kombat-3/`, y conservar las carpetas anteriores sin cambios.
